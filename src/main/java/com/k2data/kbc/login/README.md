## git拉取示例


##说明
```
1. 不拦截路径："/login"、"/register"

```
请求参数

| 属性        | 说明     | 类型   | 默认值 |
| ----------- | -------- | ------ | ------ |
| userName | 用户名 | String | Null | 
| password | 密码 | String | Null |

http://localhost:9080/login?userName=admin&password=admin
